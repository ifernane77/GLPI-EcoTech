# Plan de paramétrage GLPI — GLPI EcoTech

## Objectif

Ce document prépare les étapes de paramétrage de GLPI pour le projet EcoTech.

Il servira de guide lorsque GLPI sera installé.

---

## Étape 1 — Créer les catégories de tickets

Catégories à créer :

- Matériel ;
- Impression ;
- Réseau ;
- Cloud / Google Drive ;
- Application EcoTech ;
- Documents ;
- Sauvegarde ;
- Accès utilisateur.

---

## Étape 2 — Ajouter les équipements principaux

Équipements prévus :

| Nom GLPI | Type |
|---|---|
| PC-ADM-01 | Ordinateur fixe |
| PC-ADM-02 | Ordinateur fixe |
| PC-GER-01 | Ordinateur portable |
| PC-GER-02 | Ordinateur portable |
| TEL-PRO-01 | Téléphone professionnel |
| TEL-PRO-02 | Téléphone professionnel |
| TEL-PRO-03 | Téléphone professionnel |
| IMP-SCAN-01 | Imprimante / scanner |
| BOX-01 | Équipement réseau |

---

## Étape 3 — Créer les utilisateurs

Utilisateurs ou profils prévus :

| Profil | Utilisation |
|---|---|
| Administrateur GLPI | Configuration et gestion complète |
| Technicien support | Traitement des tickets |
| Demandeur | Création et suivi de tickets |
| Gérant | Suivi des demandes importantes |

---

## Étape 4 — Créer les tickets de test

Tickets prévus :

1. PC administratif lent ;
2. imprimante / scanner indisponible ;
3. accès à un dossier Google Drive ;
4. problème d’accès à l’application EcoTech ;
5. téléphone professionnel abîmé ;
6. devis introuvable ;
7. connexion Internet instable ;
8. mise en place d’une sauvegarde régulière.

---

## Étape 5 — Préparer la base de connaissances

Articles prévus :

- Que faire si l’imprimante ne répond plus ?
- Vérifier une connexion Internet instable.
- Retrouver un document dans Google Drive.
- Sauvegarder les documents importants.
- Réinitialiser un accès utilisateur.
- Sauvegarder les photos d’un téléphone professionnel.

---

## Étape 6 — Captures à produire plus tard

Quand GLPI sera installé, il faudra garder les preuves suivantes :

- capture des catégories créées ;
- capture de la liste des équipements ;
- capture d’un ticket incident ;
- capture d’un ticket demande ;
- capture d’un ticket résolu ;
- capture d’un article de base de connaissances ;
- capture du tableau de bord GLPI.

---

## Remarque

Ce plan de paramétrage est une préparation.

Les actions seront réalisées dans GLPI lorsque l’installation sera terminée.
