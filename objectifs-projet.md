# Objectifs du projet — GLPI EcoTech

## Objectif général

Mettre en place une simulation GLPI pour représenter la gestion du support informatique d’une petite entreprise.

## Objectifs pédagogiques BTS SIO

Ce projet doit me permettre de travailler des compétences liées à l’épreuve E5 :

- répondre aux incidents ;
- traiter les demandes d’assistance ;
- mettre à disposition un service informatique ;
- gérer le patrimoine informatique ;
- travailler en mode projet ;
- documenter des procédures.

## Objectifs techniques

Les objectifs techniques prévus sont :

- préparer une machine virtuelle Debian 11 ;
- installer les prérequis nécessaires ;
- installer GLPI ;
- créer les utilisateurs ;
- créer les équipements ;
- créer des tickets réalistes ;
- rédiger des procédures simples ;
- produire des captures exploitables.

## Objectifs professionnels

Le projet doit montrer une démarche professionnelle adaptée à une petite structure :

- analyse du parc existant ;
- identification des besoins de support ;
- centralisation des informations ;
- suivi des incidents ;
- documentation des interventions.

## Limites

Le projet reste une simulation.

Il ne prévoit pas :

- une infrastructure réseau complexe ;
- un Active Directory complet ;
- une supervision avancée ;
- une automatisation poussée.

L’objectif est de produire un dossier clair, réaliste et présentable au BTS SIO.
