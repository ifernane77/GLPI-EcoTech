# Correspondance tickets / catégories — GLPI EcoTech

## Objectif

Ce document associe les tickets prévisionnels GLPI aux catégories préparées.

Il permet de vérifier que les catégories choisies couvrent bien les incidents et demandes prévus pour EcoTech.

## Analyse

La catégorie **Matériel** est utilisée plusieurs fois car EcoTech possède plusieurs équipements : ordinateurs, téléphones professionnels et imprimante.

Les catégories **Réseau**, **Impression**, **Documents**, **Sauvegarde** et **Application EcoTech** permettent de représenter des problèmes fréquents dans une petite entreprise.

La catégorie **Accès utilisateur** n’est pas encore utilisée dans les 8 tickets initiaux, mais elle sera utile plus tard pour des demandes comme :

- création de compte ;
- réinitialisation de mot de passe ;
- modification de droits ;
- accès à l’application EcoTech ;
- accès à Google Drive.

---

## Justification BTS

Cette classification permet de montrer une démarche professionnelle de support informatique.

Elle prépare les compétences E5 :

- gérer le patrimoine informatique ;
- traiter les incidents ;
- traiter les demandes ;
- mettre à disposition un service informatique ;
- travailler en mode projet.
