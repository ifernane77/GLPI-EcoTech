# Périmètre du projet — GLPI EcoTech

## Présentation

GLPI EcoTech est un projet de simulation professionnelle permettant de représenter la gestion du support informatique d’une petite entreprise.

Le projet est basé sur le contexte de l’entreprise EcoTech, située à Saint-Pierre, à La Réunion.

## Objectif principal

L’objectif principal est de mettre en place et documenter un environnement GLPI permettant de gérer :

- le parc informatique ;
- les utilisateurs ;
- les incidents ;
- les demandes ;
- une base de connaissances simple.

## Périmètre fonctionnel

Le projet couvrira les éléments suivants :

- installation de GLPI sur Debian 11 ;
- création des utilisateurs ;
- création du parc informatique ;
- ajout de postes, téléphones et imprimante ;
- création de tickets d’incidents ;
- création de tickets de demandes ;
- rédaction de procédures simples ;
- production de captures d’écran pour le portfolio.

## Périmètre technique

Le projet utilisera :

- une machine virtuelle Debian 11 ;
- un serveur web ;
- une base de données ;
- GLPI ;
- un navigateur pour l’administration ;
- GitHub pour stocker la documentation.

## Limites du projet

Ce projet reste une simulation adaptée au niveau BTS SIO.

Il ne s’agit pas de mettre en place une infrastructure complète d’une grande entreprise, mais de montrer une démarche professionnelle réaliste pour une petite structure.

## Livrables attendus

Les livrables prévus sont :

- documentation d’installation ;
- inventaire du parc ;
- captures de GLPI ;
- exemples de tickets ;
- base de connaissances ;
- journal de bord ;
- dépôt GitHub documenté.
