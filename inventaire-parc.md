# Inventaire prévisionnel du parc informatique — EcoTech

## Objectif

Ce document prépare l’inventaire du parc informatique qui sera ensuite intégré dans GLPI.

## Matériel recensé

| Type de matériel | Quantité | Utilisation prévue |
|---|---:|---|
| PC fixes | 2 | Gestion administrative et documents |
| PC portables | 2 | Utilisation par les gérants ou employés |
| Machines Windows | 2 | Postes utilisateurs |
| Machines Mac | 2 | Postes utilisateurs |
| Téléphones professionnels | 3 | Appels, photos de chantier, échanges terrain |
| Imprimante / scanner | 1 | Impression et numérisation de documents |
| Livebox | 1 | Accès Internet de l’entreprise |

## Services et outils utilisés

L’entreprise utilise actuellement :

- Google Drive ;
- mails professionnels ;
- WhatsApp ;
- ordinateurs personnels ;
- téléphones professionnels.

## Problèmes identifiés

Les problèmes possibles sont :

- absence de centralisation complète du parc informatique ;
- absence de politique de sauvegarde clairement définie ;
- antivirus non identifié ;
- suivi des incidents peu formalisé ;
- difficulté à suivre les demandes internes ;
- documentation technique limitée.

## Apport de GLPI

GLPI peut permettre de :

- recenser les équipements ;
- suivre les incidents ;
- organiser les demandes internes ;
- garder une trace des interventions ;
- créer une base de connaissances ;
- améliorer la visibilité sur le parc informatique.
