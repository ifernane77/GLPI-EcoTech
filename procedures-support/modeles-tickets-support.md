# Modèles de tickets support — GLPI EcoTech

## Objectif

Ce document prépare des exemples de tickets qui pourront être créés plus tard dans GLPI.

Les exemples sont adaptés au contexte EcoTech.

---

## Ticket 1 — PC administratif lent

| Élément | Contenu prévu |
|---|---|
| Type | Incident |
| Catégorie | Matériel |
| Demandeur | Employé administratif |
| Priorité | Moyenne |
| Description | Le PC administratif est lent au démarrage et pendant l’utilisation. |
| Action prévue | Vérifier les programmes au démarrage, l’espace disque et l’état général du poste. |

---

## Ticket 2 — Imprimante indisponible

| Élément | Contenu prévu |
|---|---|
| Type | Incident |
| Catégorie | Impression |
| Demandeur | Employé administratif |
| Priorité | Haute |
| Description | L’imprimante / scanner ne répond plus depuis le poste administratif. |
| Action prévue | Vérifier le branchement, la connexion réseau et le statut de l’imprimante. |

---

## Ticket 3 — Accès Google Drive impossible

| Élément | Contenu prévu |
|---|---|
| Type | Demande |
| Catégorie | Cloud / Google Drive |
| Demandeur | Gérant ou employé |
| Priorité | Moyenne |
| Description | L’utilisateur ne parvient pas à accéder à un dossier partagé. |
| Action prévue | Vérifier les droits d’accès au dossier et le compte utilisé. |

---

## Ticket 4 — Document introuvable

| Élément | Contenu prévu |
|---|---|
| Type | Demande |
| Catégorie | Documents |
| Demandeur | Employé administratif |
| Priorité | Moyenne |
| Description | Un devis ou une facture n’est pas retrouvé dans les dossiers habituels. |
| Action prévue | Rechercher le document, vérifier le classement et proposer une règle de nommage. |

---

## Ticket 5 — Connexion Internet instable

| Élément | Contenu prévu |
|---|---|
| Type | Incident |
| Catégorie | Réseau |
| Demandeur | Équipe EcoTech |
| Priorité | Haute |
| Description | La connexion Internet est instable dans les bureaux. |
| Action prévue | Vérifier la Livebox, les câbles, le Wi-Fi et redémarrer l’équipement si besoin. |

## Ticket 6 — Téléphone professionnel abîmé

| Élément | Contenu prévu |
|---|---|
| Type | Incident |
| Catégorie | Matériel |
| Demandeur | Employé terrain |
| Priorité | Moyenne |

## Remarque personnelle

Ces modèles de tickets m’aideront à créer des tickets cohérents dans GLPI plus tard.

Ils sont préparés à partir du contexte réel d’EcoTech.
