# Script oral — GLPI EcoTech

## Objectif

Ce document prépare une courte présentation orale du projet GLPI EcoTech.

Il me servira plus tard pour le portfolio et l’oral.

---

## Présentation courte

GLPI EcoTech est un projet de simulation de support informatique.

Il est basé sur le contexte de l’entreprise EcoTech, une petite entreprise du BTP située à Saint-Pierre, à La Réunion.

L’objectif est de préparer un outil pour gérer le parc informatique, les incidents et les demandes internes.

---

## Besoin identifié

EcoTech utilise plusieurs équipements et outils numériques, mais il n’y a pas encore de suivi centralisé du support informatique.

Les problèmes peuvent concerner :

- les PC ;
- les téléphones professionnels ;
- l’imprimante ;
- la connexion Internet ;
- Google Drive ;
- les accès utilisateurs ;
- les documents.

---

## Solution prévue

La solution prévue est de mettre en place GLPI dans un environnement de test.

GLPI permettra de gérer :

- les équipements ;
- les utilisateurs ;
- les groupes ;
- les tickets ;
- les incidents ;
- les demandes ;
- les procédures.

---

## Ce que j’ai déjà fait

Pour l’instant, j’ai préparé la documentation du projet.

J’ai travaillé sur :

- le périmètre ;
- l’inventaire ;
- les tickets prévus ;
- les catégories ;
- le schéma du SI ;
- les flux de support ;
- les groupes et profils utilisateurs.

Je n’ai pas encore présenté GLPI comme totalement opérationnel si l’installation n’est pas terminée.
